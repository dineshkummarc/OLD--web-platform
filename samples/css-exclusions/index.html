About

CSS Exclusions brings new properties to CSS (Cascading Style Sheets) in order to provide exclusions shapes which text will wrap around.

We have put together a set of simple demos for you that show off these capabilities. Be sure to check back here regularly as we post more samples regularly.

The current set provide basic samples that demonstrate individual CSS Exclusions properties.

Runs In

Currently CSS Exclusions is not fully implemented in any browsers though we are starting to contribute the early parsing and geometry pieces to WebKit.

Check back for details on implementation progress.

In the meantime, to run the samples shared here, you will need to use a mini-browser which uses a specifically modified version of WebKit that includes support for CSS Exclusions. This experimental mini-browser will allow you to experiment with the samples, and are a way for us to explore the implementation options for these features.

To run the samples in the mini-browser, download the ZIP for Mac or Windows.

Why use CSS Exclusions

CSS Exclusions provides a great benefit to both web designers as well as developers.

With CSS Exclusions, you can take advantage of new CSS properties that let you build rich interfaces with custom text exclusions. Custom text exclusions allow text to be excluded by shapes or images

CSS Exclusions is easy to pick up. They build upon familiar CSS properties and structures. Additionally, CSS Exclusions are often used in conjunction with CSS Regions, a feature that lets you control how text flows across different regions in a layout

Basic samples

Note: To run the samples in the mini-browser, download the ZIP for Mac or Windows.

  • Content shape

    Demonstrates: custom shape for text container.

    CSS Exclusions properties used: -webkit-wrap-shape, -webkit-wrap-shape-mode

  • Text exclusion

    Demonstrates: exclusion shape affecting text layout.

    CSS Exclusions properties used: -webkit-wrap-shape, -webkit-wrap-shape-mode

  • Dynamic exclusion

    Demonstrates: text flowing around an exclusion.

    CSS Exclusion properties used: -webkit-wrap-shape, -webkit-wrap-shape-mode

  • Shape order

    Demonstrates: shapes affecting each other

    CSS Exclusions properties used: -webkit-wrap-shape, -webkit-wrap-shape-mode, -webkit-wrap-shape-order

Feedback

Please keep in mind that the CSS Exclusions prototype is a work in progress.

The development so far has been done following the CSS Exclusions editor's draft. Comments and suggestions for these documents are very welcome!

  • For comments on the CSS Exclusions editor's draft proposal please use the www-style@w3.org mailing list on the W3C website. Please use an identifier such as "[CSS Exclusions]" in the subject of your email.
  • If you have any questions or comments about the current prototype or the samples please drop us a message on the CSS Regions & Exclusions Forum.

Any feedback on the proposals is very welcome. It helps us provide a feature that's both fun and useful in the modern web landscape.
