Introducing CSSRegions

  • Create appealing digital magazines/newspapers
  • Define and arbitrarily assign flows of content between disjoint containers
  • More than one flow of content in the document -> multiple named flows
  • containers = regions -> don’t need to have the same dimensions, don’t need to have the same positioning scheme, can be styled separately

CSSRegions concepts: named flows

  • Flow: gather conceptually related content
  • Named flow: identify the flow by name
